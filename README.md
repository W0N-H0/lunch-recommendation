오늘은 뭘 먹어야 할지 고민이라면?<br>

<br>

### 점.메.추 룰렛 사용방법
----------------------------
1. 원하는 지역구를 입력해주세요.
2. 지역구를 모르신다면 우측 상단 지역구 보기를 클릭.
3. 지역구를 입력하셨다면, **룰렛 START!**
4. 룰렛에 결과에 따라 알맞은 메뉴를 추천해 드립니다.
5. 결과 보러가기 Go Go!
6. 선택하신 지역에 식당을 추천해 드립니다.
7. 자세히 보고 싶다면 상세보기를 클릭.
8. 음식 메뉴를 클릭하면 음식에 대한 상세한 내용도 볼 수 있습니다.
